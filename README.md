# 🎬 Netflix Data Cleaning Project
## 📌 Project Overview
This project demonstrates the data cleaning process on the Netflix Titles dataset using Python and Pandas.
## 📂 Dataset
- Source: Kaggle
- File: `netflix_titles.csv`
## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Google Colab
## 📋 Data Cleaning Steps
- Loaded the dataset
- Explored the dataset
- Checked data types
- Identified missing values
- Filled missing values
- Removed unnecessary missing records
- Checked duplicate records
- Exported cleaned dataset
## 📁 Files
- `Netflix_Data_Cleaning.ipynb`
- `netflix_titles.csv`
- `cleaned_netflix_titles.csv`
## 🎯 Skills Demonstrated
- Data Cleaning
- Missing Value Handling
- Data Validation
